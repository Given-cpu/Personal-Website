# Personal Introduction Website

A clean and concise personal portfolio website with a white and blue color scheme.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Clean Layout**: Minimalist design with white and blue colors
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Sections Included**:
  - Name/Hero section
  - Self-introduction/About
  - Work/Projects showcase
  - Thoughts/Reflections

## Getting Started

Simply open `index.html` in your web browser to view the website.

## Customization

1. **Name**: Edit the `<h1 class="hero-name">` in the hero section
2. **Self-Introduction**: Add your introduction in the About section
3. **Work**: Add your projects in the Work section
4. **Thoughts**: Add your thoughts and reflections in the Thoughts section
5. **Colors**: Modify the CSS variables in `styles.css` to adjust the color scheme

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Interactive features and smooth scrolling
